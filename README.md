### Prerequisites

This is an example of how to list things you need to use the software and how to install them.
* maven 3.5.2
* openjdk 17  

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/jeortizquan/groceries-knapsack-algorithm.git
   ```
2. Install Maven packages
   ```sh
   mvn install
   ```
   
3. Build app
   ```sh
   mvn package
   ```
